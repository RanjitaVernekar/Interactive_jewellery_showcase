A web-based 3D jewelry viewer that showcases elegant rings in an immersive virtual environment. Built with A-Frame for WebXR compatibility, allowing users to interact with jewelry pieces in a browser.

## Features

- **Interactive 3D Rings**: Three distinct rings displayed on a pedestal, each with unique materials and designs.
- **Click to Focus**: Click on any ring to zoom in and examine it closely.
- **Drag to Rotate**: While focused, drag to rotate the ring for a 360-degree view.
- **Elegant UI**: Minimalist interface with a branded header and subtle vignette effects.
- **Responsive Design**: Optimized for various screen sizes and devices.
- **WebXR Ready**: Compatible with VR and AR devices through A-Frame.

## How to Run

1. Clone or download the project files.
2. Open `index.html` in a modern web browser (Chrome, Firefox, Safari, or Edge recommended).
3. The application will load automatically - no server required.

For the best experience:
- Use a device with WebGL support.
- Enable hardware acceleration in your browser settings.
- For VR/AR, use a compatible headset or device.

## Technologies Used

- **A-Frame**: Web framework for building VR/AR experiences.
- **HTML5**: Structure and markup.
- **CSS3**: Styling and layout.
- **JavaScript**: Interactivity and custom components.
